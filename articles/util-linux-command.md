# カレンダーの月の英語表記を取得
## 実行コマンド
```bash
$ LANG=C cal -y | grep -Eo '[A-Z][a-z]{2,}'
```

## 出力結果
:::details クリックして表示

February
March
April
May
June
July
August
September
October
November
December
:::

## コメント
ロケール設定がポイント。
